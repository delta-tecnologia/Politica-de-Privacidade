Política de Privacidade

Este aplicativo foi desenvolvido para permitir que pais acompanhem notas e faltas de seus filhos por meio de um sistema de ensino seguro. O app se conecta a uma API interna, que por sua vez acessa dados de outro sistema de ensino pertencente à mesma empresa. Valorizamos a privacidade dos usuários e aplicamos boas práticas de segurança em todas as etapas.

Coleta e Uso de Dados

O aplicativo não coleta nem armazena informações pessoais localmente. Todos os dados exibidos no app são fornecidos temporariamente pela API, que acessa o sistema de ensino da empresa. Ao excluir o aplicativo, nenhum dado permanece no dispositivo.

O app é usado exclusivamente para visualizar informações acadêmicas do aluno, como notas e faltas, de forma temporária e segura.

Permissões do Dispositivo

O app pode solicitar permissões do sistema operacional (como acesso à internet e armazenamento temporário) apenas para garantir seu funcionamento correto. Nenhuma permissão é utilizada para coleta de dados pessoais.

Uso de Dados de Terceiros

O aplicativo não compartilha dados com terceiros. Todas as informações permanecem dentro do ecossistema da empresa, entre o app, a API e o sistema de ensino. Nenhum dado é usado para fins comerciais ou repassado a serviços externos.

Segurança

A segurança dos dados é uma prioridade. Embora o app não armazene informações localmente, seguimos boas práticas de proteção de dados, incluindo:

  • Comunicação com a API via protocolos seguros (HTTPS);
  • Controle de acesso à API para garantir que apenas usuários autenticados possam visualizar dados;
  • Armazenamento temporário apenas na memória do app durante a sessão;
  • Boas práticas de desenvolvimento seguro para prevenir vulnerabilidades e garantir a estabilidade do aplicativo.
