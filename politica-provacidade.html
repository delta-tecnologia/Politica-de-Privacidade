```html Política de Privacidade

Política de Privacidade

Este aplicativo foi desenvolvido para oferecer uma experiência simples e segura. Respeitamos sua privacidade e não coletamos informações pessoais dos usuários.

Coleta de Dados

Este aplicativo não coleta, armazena ou compartilha dados pessoais identificáveis, como nome, e-mail, telefone ou localização.

Permissões do Dispositivo

O aplicativo pode solicitar permissões do sistema operacional (como acesso à internet, armazenamento ou outros recursos) apenas para o funcionamento correto do app, nunca para coleta de dados pessoais.

Uso de Dados de Terceiros

Este aplicativo não utiliza serviços de terceiros que coletem informações de forma automática.

Segurança

Mesmo não coletando dados pessoais, seguimos boas práticas para manter o aplicativo seguro e estável.

Alterações nesta Política

Podemos atualizar esta política periodicamente. A versão mais recente estará sempre disponível através deste link.

Contato

Se tiver dúvidas ou sugestões sobre esta Política de Privacidade, entre em contato pelo e-mail: jvieiramelo@deltatecnologia.net.br
```
